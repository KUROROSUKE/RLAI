ゲームルール

1. 手札の元素を使って化合物を作ろう！

2. 交換したい場合は、クリックしてカードを捨てよう。

3. 生成できる化合物があれば、「精製」または「融合」ボタンを押した後にカードを選択して作ろう！

4. 相手と250ポイント以上の差がつくか、10ゲームが終わった時点でポイントの多い方が勝利！
